\begin{abstract}
Im ersten Teil dieser Arbeit wird beschrieben, wie eine auf Eclipse basierende Entwicklungsumgebung für MicroCore implementiert wurde. Im zweiten Teil werden verschiedene Peephole Optimierungen, unter anderem der vom Davidson und Fraser beschriebene Optimierer PO, für den Compiler untersucht und mit der aktuellen Peephole Implementation des Forth-Cross Compilers verglichen.
\end{abstract}
